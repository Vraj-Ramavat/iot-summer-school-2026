# 4-Key Digital Piano using Buzzer (Q15)

## Hardware Required
- 1x Arduino Uno
- 5x Push Buttons (4 Notes + 1 Mode Toggle)
- 5x 10kΩ Resistors (for button pull-downs)
- 1x Passive Buzzer

## Circuit Connections
1. Buttons 1, 2, 3, 4 connected to pins 2, 3, 4, 5 respectively (with 10kΩ pull-downs to GND).
2. Mode Toggle Button connected to pin 6 (with 10kΩ pull-down to GND).
3. Passive Buzzer positive pin connected to pin 11, negative pin to GND.

## Expected Output
Pressing button 1, 2, 3, or 4 plays notes Do, Re, Mi, or Fa respectively. Releasing silence the buzzer. Pressing two or more notes simultaneously plays Sol. Pressing the Mode button toggles between Major and Minor scale frequencies.
