# IoT Summer School 2026 - Hands-On Learning

**Student Name:** Vraj Ramavat  
**Course:** Internet of Things – Hands-On Learning  

## Project Description
This repository contains a comprehensive set of hands-on assignments and project challenges completed during the IoT Summer School 2026. It features Arduino programming sketches, sensor and actuator integrations, state machine implementations, and version-controlled firmware documentation. The codebase demonstrates key IoT concepts including non-blocking multitasking, physical computing, and local edge data logging.

---

## Directory Structure
The repository is structured into weekly modules matching the course syllabus:
- `/week1` - GitHub Setup & Version Control (Git workflow, branching, CI/CD actions, theory)
- `/week2` - IoT Fundamentals & Arduino Programming (traffic controller, serial interface, fading, state machines, theory)
- `/week3` - Sensors, Actuators & Data Acquisition (DHT11, LDR, HC-SR04, servo, DC motor, relay, LCD, theory)
- `/week4` - Wireless Communication & Protocols (Placeholder for wireless modules)
- `/week5` - Advanced IoT Projects & Problem Statements (Placeholder for final project ideas)
- `/final-project` - Final Capstone Project codebase and resources

## Technical Stack & Tools
- **Hardware Platform:** Arduino UNO, ESP32/ESP8266 (Simulated/Physical)
- **Programming Language:** Embedded C++ (Arduino sketch `.ino`), Python (helper scripts)
- **Simulation Environments:** Tinkercad, Wokwi
- **Version Control:** Git & GitHub Actions (CI/CD)

For specific instructions and build details, please navigate to the respective subdirectories.
